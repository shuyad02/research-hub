# ResearchHub AI

**ResearchHub AI** is an enterprise-grade full-stack platform designed to help students, researchers, and professionals manage research projects, discover international scholarships, and collaborate effectively. Powered by Google Gemini AI, the platform acts as an intelligent assistant providing tailored recommendations and research insights.

## Key Features
- **Role-Based Portals**: Distinct dashboards for Students, Researchers, and Admins.
- **Intelligent Scholarship Discovery**: AI-driven eligibility checker based on academic profiles.
- **Research Assistant**: Generate research proposals and roadmaps using Gemini AI.
- **Collaboration System**: Form teams, manage documents, and track research project progress.
- **Enterprise Security**: JWT-based authentication with robust access controls.

## Tech Stack
- **Frontend**: Next.js 16, React 19, TypeScript, Tailwind CSS, Shadcn UI
- **Backend**: Java 21, Spring Boot 3, Spring Security, Hibernate
- **Database**: PostgreSQL
- **DevOps**: Docker, Docker Compose, GitHub Actions
